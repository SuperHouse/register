# User Guide

This guide covers the web interface of the Testomatic Register.

All pages require login. What you see depends on your role:

- **Staff users** see all data across all organisations and have access to editing, admin tools, and internal notes.
- **Organisation users** see only data belonging to their associated organisation, with internal records hidden.

## Navigation

The left sidebar provides links to the main sections:

| Section | Description |
|---|---|
| [Dashboard](dashboard.md) | Summary statistics and assembly chart |
| [Batches](batches.md) | Production run tracking |
| [Boards](boards.md) | Individual PCBs by serial number |
| [Designs](designs.md) | Board types and their associated files |
| [Organisations](organisations.md) | Customers and their users |
| [Parts](parts.md) | Component library with sourcing and pricing |
| Settings | Production stages, locations, BOM import rules |

The current application version is shown at the bottom of the sidebar and links to the source repository.
